# TimeLog #
App to track work easily

##Requirements:##
* Python 3.5
* Postgres
* Django 1.10
* Nodejs
* Gulp

## Install Npm packages ##
```
cd static/
npm install
```

## Installing Gulp ##
```
cd static/
npm install gulp -g
```

## Build Gulp ##
compile the html, es6, sass and minify
```
cd static
gulp build
```

## Clean dist folder ##
```
cd static/
gulp clean
```

## Watch file changes ##
```
gulp watch
```
